This package (sosutl11) should contain the following files:

    ReadMe.1st          This file
    License             GNU General Public License (version 1.0); see
                        `Copyrights' section below

    sostree.exe         Directory tree drawer
    sostree.man         Manual page for SOS Tree
    sostree.c           Source code for SOS Tree (BC 1.0)

    chname.cmd          Changes HPFS names to FAT format and vice versa
    chname.man          Manual page for ChName

    extract.cmd         Extract files from / List contents of archives
    extract.man         Manual page for Extract

    findfile.cmd        File finder
    findfile.man        Manual page for FindFile

    ps.cmd              Process Status simplifier utility
    ps.man              Manual page for ps

All these programs are public domain.

Notes ----------------------------------------------------------------

I'd be happy to obtain versions of these programs in different
languages. If you are able to translate the program messages, please
send the modified sources to me so that I can include them in an
update package of SOS Utilities. NOTE: EVEN IF YOU DON'T HAVE ACCESS
TO BORLAND C++ COMPILER you still can translate the messages in the
source files, and then send the sources to me. I'll distribute the
recompiled programs further.
